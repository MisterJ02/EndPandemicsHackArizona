## 🚀 Features

- 🗺️ Interactive map for adding/viewing outbreak reports
- 📍 Location-based risk calculation
- 📊 Risk score + breakdown (low / medium / high severity)
- 🧠 AI-generated health analysis using Google Gemma
- 📰 Integrated WHO + CDC outbreak news (with tab switching)
- 📡 Real-time updates via polling
